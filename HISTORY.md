# History

This file tracks the conceptual and methodological evolution of the
aXIOM project. It is not a personal-genesis narrative.

## Timeline

- **2026 Q1.** Initial framing as an integrated AI-assisted academic
  assessment platform with a methodological apparatus.
- **2026-04-14.** Pre-pilot compliance audit completed. Three
  critical blockers identified prior to institutional pilot.
- **2026-04-20.** DFS (Declarative Frame Schema) named and recognised
  as the load-bearing methodological IP of the project — distinct
  from the surrounding software shell.
- **2026-05-04.** Privacy and security audit merged
  (`SECURITY_AUDIT.md`). Four high-severity findings documented;
  remediation deferred.
- **2026-05-12.** Institutional pilot deferred.
- **2026-05-18.** Scope-cut. Methodology track and product/pilot
  track formally decoupled. See `STATUS.md`.

## Note on pre-2026-05-18 documents

Documents in this repository predate the scope-cut unless their
header note states otherwise. They are preserved as historical
record. For the current track split see `STATUS.md`.
